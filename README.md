# 📁 Proyectos en Desarrollo

Repositorio que contiene varios proyectos personales en diferentes estados de desarrollo. Aquí encontrarás aplicaciones en **Python**, **JavaScript/HTML/CSS** que demuestran habilidades en programación orientada a objetos, desarrollo web frontend y gestión de datos.

---

## 📂 Estructura de Proyectos

### 1. **🛒 Sistema de Tienda - Final** (Python - En Progreso)
Aplicación de gestión de tienda con carrito de compras, tarjetas de pago y productos.

**Características:**
- Gestión de clientes y sus datos personales
- Catálogo de productos con precios y categorías
- Sistema de carrito de compras
- Procesamiento de pagos con validación de tarjetas
- Módulo de vehículos de entrega
- Interfaz de menú interactivo

**Tecnologías:** Python 3, POO (Programación Orientada a Objetos)

**Archivos clave:**
- `ClasePersona.py` - Modelo base de personas
- `ClaseProducto.py` - Gestión de productos
- `ClaseTarjeta.py` - Validación y procesamiento de tarjetas
- `ClaseTienda.py` - Lógica principal de la tienda
- `Menu.py` - Interfaz de usuario

---

### 2. **🎓 Sistema de Gestión Universitaria - GestionUniv** (Python - En Progreso)
Aplicación para gestionar estudiantes, profesores y materias en una institución educativa.

**Características:**
- Registro y gestión de estudiantes
- Información de profesores
- Catálogo de materias
- Asignación de materias a estudiantes
- Jerarquía de clases con herencia
- Estructura modular de la aplicación

**Tecnologías:** Python 3, POO, Herencia de clases

**Módulos principales:**
- `clasePersona.py` - Clase base para personas
- `claseAlu.py` - Gestión de estudiantes
- `claseProfe.py` - Gestión de profesores
- `claseMater.py` - Catálogo de materias
- `claseUniv.py` - Lógica central de la universidad

---

### 3. **🎬 Frontend Netflix - FrontNetflix** (HTML/CSS/JavaScript - En Progreso)
Interfaz estilo Netflix con catálogo de contenido (películas y series).

**Características:**
- Diseño responsivo tipo Netflix
- Catálogo de películas y series
- Interfaz moderna y atractiva
- Estructura HTML semántica
- Estilos CSS avanzados

**Tecnologías:** HTML5, CSS3, JavaScript

**Estructura:**
- `index.html` - Página principal
- `Css/` - Archivos de estilos
- `Js/` - Scripts interactivos
- `Img/` - Recursos de imágenes

---

### 4. **⭐ Frontend Reseñas - FrontReseña** (HTML/CSS - En Progreso)
Página web para visualizar y escribir reseñas de películas y series.

**Características:**
- Sistema de reseñas con calificación por estrellas
- Formulario para agregar nuevas reseñas
- Diseño limpio e intuitivo
- Gestión de comentarios de usuarios

**Tecnologías:** HTML5, CSS3

**Archivos:**
- `Prueba.html` - Página principal
- `css/` - Estilos personalizados
- `Etiquetas.txt` - Referencia de etiquetas HTML

## 🚀 Cómo Usar

### Proyectos Python
```bash
# Navegar a la carpeta del proyecto
cd Final
# o
cd GestionUniv

# Ejecutar la aplicación
python Menu.py
# o
python aplicAlu.py
```

### Proyectos Web Frontend
```bash
# Abrir en el navegador
# Haz doble clic en index.html o Prueba.html
# O usa un servidor local:
python -m http.server 8000
# Luego accede a http://localhost:8000
```
